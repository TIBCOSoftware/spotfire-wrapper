Demo Spotfire™ Wrapper - Two dashboards

Several dashboards can be inserted. Only one script tag needs to be placed after all spotfire-wrapper tags:

Code:

<spotfire-viewer
    url="https://spotfire-next.cloud.tibco.com"
    path="Sample Data Functions/MATLAB/MATLAB_Correlation_Cars">
    page="The Analysis"
</spotfire-viewer>

<spotfire-editor
    url="https://spotfire-next.cloud.tibco.com"
    path="Samples/Expense Analyzer Dashboard" 
    page="Expense dashboard">
</spotfire-editor>

<script src="./spotfire-wrapper.js" ></script>
