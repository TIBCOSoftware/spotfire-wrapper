Demo Spotfire™ Wrapper - Two dashboards in same page

Previous demo (Basic sample) | Next demo (Customize dashboard)

Several dashboards can be inserted. Only one script tag needs to be placed after all spotfire-wrapper tags:

Code:

<spotfire-wrapper
    url="https://spotfire-next.cloud.tibco.com/spotfire/wp"
    path="Samples/Sales and Marketing" 
    page="0">
</spotfire-wrapper>

<spotfire-wrapper
    url="https://spotfire-next.cloud.tibco.com/spotfire/wp"
    path="Samples/Expense Analyzer Dashboard" 
    page="Expense dashboard">
</spotfire-wrapper>

<script src="https://s3-us-west-2.amazonaws.com/cec-library/spotfire-wrapper.js" type="text/javascript"></script>
