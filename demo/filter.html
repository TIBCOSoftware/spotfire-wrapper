Previous Demo (Customize dashboard) | Next Demo (get Marking)

filters attribute is used to set a filters that will be applied when dashboard opens:

Code:

<spotfire-wrapper
    url="https://spotfire-next.cloud.tibco.com"
    path="Samples/Sales and Marketing" 
    filters='[{"dataTableName": "Supermarket Stock excel - Sheet1", "dataColumnName": "Type", "filterSettings": { "values": ["Beverages", "Meat"]}}]'
    page="Sales Performance">
</spotfire-wrapper>

<script src="https://s3-us-west-2.amazonaws.com/cec-library/spotfire-wrapper.js" type="text/javascript"></script>
filters is an array made of 
  [{
    "dataTableName": "<data table name>",
    "dataColumnName": "<data column name>"
    "filterSettings": { <filter settings>" }
  }, ... {
    "dataTableName": "<data table name>",
    "dataColumnName": "<data column name>"
    "filterSettings": { <filter settings>" }
  }]
