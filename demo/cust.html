Demo Spotfire™ Wrapper - Customize dashboard

User can customize the dashboard using customization attribute:

Code:

<spotfire-editor
    url="https://spotfire-next.cloud.tibco.com"
    path="Samples/Sales and Marketing"
    customization='{"showAuthor":true, "showFilterPanel":true, "showToolBar":true}'
    page="0">
</spotfire-editor>

<script src="./spotfire-wrapper.js" ></script>
customization is an object containing theses settings (set to false if not specified): 
  "showAbout": true
  "showAnalysisInformationTool": true
  "showAuthor": true
  "showClose": true
  "showCustomizableHeader": true
  "showDodPanel": true
  "showExportFile": true
  "showExportVisualization": true
  "showFilterPanel": true
  "showHelp": true
  "showLogout": true
  "showPageNavigation": true
  "showAnalysisInfo": true
  "showReloadAnalysis": true
  "showStatusBar": true
  "showToolBar": true
  "showUndoRedo": true
